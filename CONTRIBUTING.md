# 贡献指南

感谢你对《魔域团》设计文档的关注！

## 项目状态

本项目为**设计归档**，不包含可运行的游戏代码。  
游戏本身不再开发，文档目前处于冻结状态，但欢迎：

- 对设计内容进行澄清或修正
- 补充你发现的逻辑漏洞或边界情况
- 翻译成其他语言
- 扩展新模块或玩法设想（可在 Issue 中讨论）

## 如何贡献

1. Fork 本仓库
2. 在 `DESIGN.md` 或对应文件中进行修改
3. 提交 Pull Request，说明修改原因和内容
4. 维护者审核通过后即可合入

本仓库目前没有严格的自动化检查，请尽量保持格式一致，并使用 Markdown 标准语法。

## 行为准则

请遵守常见的开源社区行为规范，友善交流。
