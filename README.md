# 魔域团 - 完整游戏技术设计文档

一款未实际开发的移动端 10 人实时联机游戏《魔域团》的**完整技术设计参考**。

## 关于本项目

- 这份文档包含前端 Canvas 渲染规范、后端 Node.js 高并发房间架构、防作弊验证、完整数值公式与规则数据
- 设计严格遵循**服务器权威**原则，前端仅负责输入采集和画面渲染
- 所有模块职责、消息协议、数据库表结构、部署方案均已明确

## 如何使用

- 想了解一个 **移动端 Moba-Roguelike** 游戏从零到一的设计思路？直接阅读 `DESIGN.md`
- 需要作为你项目的参考文档？直接复用其中的架构或数值表格即可，遵守 MIT 许可证
- 想补充或修正某部分设计？欢迎提 Issue 或 PR，详见 `CONTRIBUTING.md`
